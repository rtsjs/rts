# rts

## Hacking RTS algorithms in node
##Getting Started

1. Clone this repo

1. Run `npm install`
1.1 Downgrade npm:  `npm install -g npm@3.3.4` . With 3.3.6 nodemon install fails.

1. To run build, node/express host, monitor `*.ts` in one step use: `gulp serve-dev`
